Embedding Effects in JavaScript

JavaScript language keeps extending with new side effects. The recent ones are Generators and Async functions. But there is a problem. It is impossible to add another useful side effect without changin the language.... Read more

This transpiler offers unlimited JavaScript extensions with new Monad interface compatible side effects loaded as a runtime library, not even as compiler plugin. There are a few Monads already implemented and usable for many practical applications.

Generated code is human readable and executable for most JavaScript engines including browsers. The tool translates ES5 to ES5. So it may embed arbitrary new side effects into other languages with compiler targeting JavaScript, like Babel, CoffeeScript, TypeScript, etc.
